`NSDateComponents` 用于表示日期的组成部分（年/月/日/时/分/秒等）。通常与 `NSCalendar`（如可用）配合用于构造/分解日期。

## 实例成员 (Instance members)

### 常用属性

- `year` / `month` / `day`
- `hour` / `minute` / `second`
- `weekday` / `weekdayOrdinal`
- `weekOfYear` / `weekOfMonth`
