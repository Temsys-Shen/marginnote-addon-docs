---
title: UIRotationGestureRecognizer
description: 旋转手势识别器。支持读取/设置 rotation，并读取 velocity。
---

`UIRotationGestureRecognizer` 继承自 [UIGestureRecognizer](/reference/uikit/uigesture-recognizer/)，用于识别双指旋转手势。

## 实例成员 (Instance members)

### 属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `rotation` | number | 当前旋转角度（弧度，可读写）。 |
| `velocity` | number | 只读。旋转速度（弧度/秒）。 |
