`NSTextStorage` 是富文本存储类型，常出现在 `UITextView` 等文本系统中，用于承载可变富文本并触发相关更新。

## 实例成员 (Instance members)

`NSTextStorage` 默认由 `UITextView.textStorage` 获取并间接使用。若需要定位某个成员名，用站内搜索。

## 类成员 (Class members)

当前接口未额外暴露 `NSTextStorage` 的类成员。

## 提示

- 在插件中默认通过 `UITextView` 等组件间接使用。
- 具体可用成员使用站内搜索成员名。
