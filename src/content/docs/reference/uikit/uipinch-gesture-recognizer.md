`UIPinchGestureRecognizer` 继承自 [UIGestureRecognizer](/reference/uikit/uigesture-recognizer/)，用于识别捏合缩放手势。

## 实例成员 (Instance members)

### 属性

| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `scale` | number | 当前缩放比例（可读写）。 |
| `velocity` | number | 只读。缩放速度。 |
